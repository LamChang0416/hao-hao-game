# 澔澔遊戲大世界 🌈 (HaoHao Game World)

這是一套專為澔澔打造的高品質 Web 學習遊戲系列，結合了 **Matter.js** 物理引擎與 **Three.js** 3D 視覺特效！

## 🚀 立即開始遊戲
您可以透過 GitHub Pages 直接在線玩這些遊戲：
[https://LamChang0416.github.io/hao-hao-game/](https://LamChang0416.github.io/hao-hao-game/)

## 🎮 遊戲內容
1. **星際數字探險 (100.html)**：在星際中尋找能量數字。
2. **平衡吧！阿布 (balance.html)**：物理平衡挑戰，抓星星。
3. **阿布 3D 魔術方塊 (logic.html)**：3D 空間建構挑戰。
4. **阿布超市小星探 (math.html)**：趣味物理購物與數數練習。
5. **阿布太空巡邏隊 (shoot.html)**：重力彈弓拋物線射擊。
6. **阿布時光探險家 (time.html)**：3D 時空隧道與時鐘訓練。

## ✨ 技術亮點
- **物理擬真**：使用 Matter.js 實現真實的碰撞與重力感。
- **3D 視覺**：使用 Three.js 提供沉浸式的 3D 背景與門戶特效。
- **專業引導**：整合 Web Speech API 提供溫柔的語音教學與鼓勵。
- **現代介面**：採用 Tailwind CSS 與 Glassmorphism 毛玻璃設計。
